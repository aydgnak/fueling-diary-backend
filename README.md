## Requirements
* NodeJS (>=18.18.2)

## Installation

```bash
# install dependencies
$ yarn install

# copy .env file
$ cp .env .env.local 
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
